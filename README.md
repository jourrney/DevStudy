## 📖 커밋 규칙

| Prefix      | Meaning                     |
|-------------|-----------------------------|
| `feat:`     | 새로운 기능 추가              |
| `fix:`      | 버그 수정                    |
| `refactor:` | 코드 리팩토링                |
| `docs:`     | 문서 수정 (README 등)        |
| `chore:`    | 기타 변경사항 (패키지, 설정 등) |

> 예시: `feat: add TodoInput component`
